شاید براتون پیش آمده باشه که بخواهید کانفیگ فیلترشکن رو از طریق پیامک به دوستتون ارسال کنید ( خصوصا" زمانی که اینترنت ضعیف هست یا مختل ) ؛ اگر به صورت عادی ارسال کنید طبیعتا" یا ارسال نمیشه یا ارسال و فیلتر شدن اون یکی هست ، اگر هم بخواهید رمزنگاری شده ارسال کنید قطعا" حساسیت اپراتور و دستگاههای نظارتی رو سمت خودتون جلب میکنید ، حالا این پروژه چیکار میکنه ، کل متن رو تبدیل میکنه به ایموجی !

حالا اگر لازمه برای دوستتون پیام ارسال کنید میتونید خروجی که فقط شامل ایموجی هست رو لابه لای چند پیام قرار بدید و ارسال کنید ، دوست شما هم بایستی از ابتدا ایموجی ها رو کپی ( از متن اصلی جدا کنه ) و به Decoder بده تا خیلی راحت به کانفیگ برسه ؛ اینجوری هم کانفیگ در امانه هم پیام شما به مقصد رسیده.
